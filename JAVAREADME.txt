Before using this program make sure you have the following prerequisites: 
1. Java Version 1.6.*+ Installed. (Refer to getting_started_java.docx)
2. Unix Based OS Machine: Fedora 19, MacOSX
3. Make sure the files you'll want to transfer is in the same directory as your Java program



Run the following in sequence, Server then Client.

1. [RUNNING IN TERMINAL - Server]
**IMPORTANT**  
cd ~/JavaServer
java -jar "JavaServer.jar"


2. [RUNNING IN TERMINAL - Client]
cd ~/JavaClient
java -jar "JavaClient.jar"

Wait for the client to prompt you.
